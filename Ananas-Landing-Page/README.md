# 🍍 Ananas 랜딩 페이지 

> **Z세대를 위한 맞춤형 메신저 및 커뮤니티 서비스 'Ananas'의 소개 웹페이지입니다.** ><br>
사용자의 피드백을 반영한 앱의 주요 기능(3D 미니룸, 릴스 메이커, 보이스룸 등)을 직관적으로 소개하고, <br>앱 다운로드를 유도하는 랜딩 페이지입니다.

---

## 🚀 주요 특징 (Key Features)

* **Z세대 맞춤형 UI/UX**: 파스텔톤과 노란색을 메인 컬러로 활용한 트렌디한 디자인
* **반응형 요소 배치**: 상단 서비스 화면 슬라이더 구조 및 하단 기능 카드 레이아웃
* **사용자 친화적 가이드**: 실제 사용자(초등학생/중학생)의 후기 섹션 및 출석 보상 시스템 시각화

1. **채팅방 및 3D 미니룸**: 친구들과 실시간 소통 및 공간 꾸미기
2. **놀이터 및 프로필**: 게임, 퀴즈, 음성방 참여 및 개인 아바타 설정

## 🛠️ 기술 스택 (Tech Stack)

* **HTML5**: 웹페이지의 구조 정의 및 콘텐츠 배치
* **CSS3**: Flexbox/Grid 레이아웃을 활용한 정렬, 컴포넌트 스타일링, 메인 컬러 시스템 구축
