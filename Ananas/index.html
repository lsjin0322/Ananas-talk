HOW TO USE

다른 사람과 함께 채팅하는 방법

01

서버 실행

npm start로 서버를 켜고
http://서버IP:4000으로 접속

→
02

아지트 개설

방장이 [톡 시작] → 아지트 만들기로
6자리 초대 코드 발급

→
03

친구 초대

친구에게 공유

→
04

함께 대화

같은 방 코드로 입장하면
실시간 대화 시작

원격/팀 사용 (DAO 등): 서버를 외부 접근 가능한 IP에서 실행하고, http://[서버IP]:4000 주소를 팀원에게 공유하세요. 또는 ngrok을 사용해 로컬 서버를 외부에 노출할 수 있습니다: ngrok http 4000

01

초저지연 실시간 채팅

WebSocket 기반으로 메시지가 전송되는 순간, 모든 참여자에게 동시에 도착합니다.

02

방 코드 초대

6자리 고유 코드 + 링크로 원하는 사람만 초대.

03

메시지 검색

채팅방 내 키워드 실시간 검색, 하이라이트 표시.

04

공지 고정

방장이 중요 메시지를 상단 고정. 전원에게 표시.

05

사진 공유

JPG·PNG·WEBP 이미지를 채팅에서 바로 전송.

06

온라인 인원

지금 접속 중인 멤버 수 실시간 표시.

Claude AI와 직접 대화. 질문, 번역, 아이디어 제안 등 다양하게 활용.

BRAND STORY

아나나스,
파인애플처럼.

파인애플(Ananas)은 겉은 거칠어 보여도 안에는 달콤함이 가득합니다. 아나나스 메신저도 마찬가지예요 — 단순한 채팅 앱이 아니라, 우리만의 따뜻한 아지트.

CONCEPT

우리만의 공간에서 자유롭게 대화하고 싶다는 아이디어에서 시작

DESIGN

픽셀아트 파인애플 캐릭터로 귀엽고 따뜻한 브랜드 아이덴티티 구축

BUILD
