## 1. 프로젝트 개요
영유아의 뇌 발달을 돕기 위해 제작된 웹 기반 교육용 게임입니다. 
집중력 및 인지 능력, 분류, 리듬감, 수 감각을 향상시킬 수 있는 4가지 미니 게임으로 구성되어 있습니다. 
시각적 과자극을 방지하는 크림색 배경과 영아 피질 활성화를 돕는 고채도 원색 중심의 색상 팔레트를 적용했습니다.

## 2. 주요 기능 및 게임 소개
* **색깔 기차 (순서·기억):** 큐(Queue) 구조를 응용하여 제시되는 기차 칸의 색깔 순서를 기억하고 순서대로 맞추는 게임입니다.
* **동물 나라 (분류·언어):** 딕셔너리(Dictionary)와 큐(Queue) 구조를 응용하여 동물의 서식지를 바다, 육지, 하늘 중에서 분류하는 게임입니다.
* **박자 놀이 (리듬·청각):** 스택(Stack)과 큐(Queue) 구조를 응용하여 점진적으로 누적되는 박자 패턴을 기억하고 순서대로 입력하는 게임입니다.
* **과일 바구니 (수 감각):** 우선순위 큐(Priority Queue) 원리를 응용하여 여러 종류의 과일 개수를 센 뒤, 개수가 적은 것부터 오름차순으로 정렬하는 게임입니다.

## 3. 기술 스택
* HTML5
* CSS3 (CSS 변수, Flexbox, Grid 레이아웃, 키프레임 애니메이션 적용)
* Vanilla JavaScript

## 4. 실행 방법
별도의 설치나 서버 구성 없이 `index.html` 파일을 웹 브라우저에서 열어 즉시 실행할 수 있습니다.
