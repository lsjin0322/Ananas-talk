## ✨ 핵심 기능
- **할 일 추가:** 입력창에 할 일을 적고 바로 아래 '추가' 버튼을 누르거나 `Enter` 키를 누르면 목록에 등록됩니다.
- **할 일 완료 토글:** 리스트의 항목을 클릭하면 취소선(line-through) 효과와 함께 완료 상태로 전환됩니다.
- **할 일 삭제:** 우측의 직관적인 붉은색 '삭제' 버튼을 누르면 해당 항목이 리스트에서 완전히 제거됩니다.
- **예외 처리:** 빈 칸인 상태로 추가를 시도하면 알림창을 띄워 잘못된 입력을 방지합니다.
