# 🌍 About Project

Travel Buddy는 여행 준비부터 여행 중 관리까지  
하나의 앱에서 해결할 수 있도록 기획한 서비스입니다.

단순 일정 앱이 아니라,

- AI 일정 생성
- 여행 가계부
- 실시간 번역
- 여행 체크리스트
- 여행 기록

등을 연결하여  
더 편하고 감성적인 여행 경험을 만드는 것을 목표로 하고 있습니다.

# ✨ Main Features

<div align="center">

| Feature | Description |
|---|---|
| 🧠 **AI Travel Planner** | 사용자의 여행 스타일과 기간에 맞춰 AI가 여행 일정을 추천하여 맞춤 생성 |
| 💰 **Travel Expense Tracker** | 여행 중 사용한 경비를 보다 빠르고 정확하게 기록하고 관리 |
| 🌐 **Real-time Translator** | 여행에서 자주 사용하는 문장을 오프라인으로 빠르게 번역 |
| 🗓 **Day Timeline** | AI가 여행 일정을 시간대별로 정리하여 하루의 여행 흐름을 한눈에 볼 수 있도록 도와주는 기능 |
| ☁️ **Buddy Help** | 환전 정보, 응급 상황, 병원 위치, 현지 팁 등 여행 중 필요한 정보를 빠르게 확인할 수 있도록 도와주는 기능 |
| 📊 **Personal Travel Data** *(Planned)* | 추후 사용자의 여행 데이터를 저장하고 개인 맞춤 기능으로 확장 예정 |

</div>

# 📌 Goal

Travel Buddy는 단순한 여행 앱이 아니라,

여행을 어렵게 느끼는 사람들도  
더 쉽고 편안하게 여행을 준비하고 즐길 수 있도록 돕는 서비스를 목표로 하고 있습니다.

또한 언어의 장벽이나 현지 정보 부족으로 인해  
여행에 어려움을 겪는 상황을 줄이고,

번역, 일정 관리, 여행 지원 기능 등을 통해  
누구나 조금 더 자유롭고 부담 없이 새로운 곳으로 떠날 수 있는 경험을 만드는 것을 지향합니다.

Travel Buddy는 여행자의 이동과 경험을 함께하며,  
실질적인 도움과 편리함을 제공하는 여행 동행 서비스를 목표로 합니다.
