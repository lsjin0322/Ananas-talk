## 🎮 주요 기능 및 디자인 설명

### 1. 디자인 컨셉: "Gym as a Dungeon"
- **컨셉**: 헬스장을 단순히 운동하는 공간이 아닌, 자신의 캐릭터를 육성하는 **'던전 대기실'**로 시각화했습니다.
- **UI/UX**: `backdrop-filter: blur()`를 활용한 **글래스모피즘(Glassmorphism)** 기법으로 세련된 반투명 카드 UI를 구현했습니다.
- **색상 전략**: 심해의 어두운 남색과 대비되는 네온 블루, 핑크를 사용하여 미래지향적인 사이버펑크 스타일을 강조했습니다.

### 2. 핵심 구현 로직
- **커스텀 모달(Modal) 시스템**: 브라우저 기본 알림창 대신 CSS 애니메이션이 적용된 커스텀 팝업을 제작했습니다. 자바스크립트를 이용해 특정 이벤트 발생 시에만 `display` 속성을 제어하여 몰입감을 높였습니다.
- **인터랙티브 요소**: 버튼 호버 시 발생하는 네온 글로우 효과와, 클릭 시 부드럽게 등장하는 `@keyframes` 애니메이션을 통해 게임 같은 사용자 경험을 제공합니다.
- **폼 유효성 제어**: 사용자가 필수 정보를 모두 입력했는지 확인하고, 데이터의 무결성을 유지할 수 있도록 설계했습니다.

## 🛠 사용 기술
- **Frontend**: HTML5, CSS3, JavaScript
- **Fonts**: Orbitron, Noto Sans KR (Google Fonts)

## 🚀 실행 방법
1. 이 저장소를 클론(Clone)하거나 ZIP으로 다운로드합니다.
2. `index.html` 파일을 크롬(Chrome) 등 브라우저로 실행합니다.
3. 플레이어 정보를 입력하고 **[모험 시작]** 버튼을 눌러 던전에 입장하세요!
