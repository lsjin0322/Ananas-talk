# 🚗 NEON ROAD - Mini Driving Survival Game

NEON ROAD는 HTML5 Canvas와 JavaScript로 제작된 2D 드라이빙 미니게임입니다.  
플레이어는 차량을 조작하여 도로 위 장애물을 피하고, 코인을 수집하며 최대한 오래 생존하는 것이 목표입니다.

단순한 러너 게임이 아니라  
생존, 점수 성장, 스테이지 시스템이 결합된 캐주얼 아케이드 게임입니다.

# 🧠 GAME SYSTEM

## 📈 Stage System
- 점수 증가에 따라 자동으로 스테이지 상승
- 스테이지가 올라갈수록 속도 증가

## ❤️ Life System
- 기본 5개의 목숨 제공
- 장애물 충돌 시 목숨 감소
- 0이 되면 게임 종료

## 💰 Score System
- 생존 시간 + 장애물 회피 + 코인 수집 기반 점수
- 콤보 유지 시 추가 점수 획득

## ⚡ Difficulty Scaling
- 시간이 지날수록 장애물 속도 증가
- 스테이지 상승 시 게임 난이도 상승
