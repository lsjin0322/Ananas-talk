##  주요 기능 및 특징

*   **반응형 레이아웃:** `clamp()` 함수와 `float` 기반의 레이아웃을 사용하여 다양한 화면 크기에 대응합니다.
*   **고정 네비게이션:** `position: sticky`를 사용하여 스크롤을 내려도 메뉴바가 상단에 고정되어 편리한 이동이 가능합니다.
*   **스무스한 사용자 경험:** 네비게이션 메뉴 클릭 시 해당 섹션으로 부드럽게 이동하며, `:target` 선택자를 활용해 선택된 섹션에 강조 효과를 주었습니다.
*   **감성적인 디자인:** 벚꽃을 연상시키는 보라색(`HEX #8E7C9C`)과 분홍색(`HEX #D17A98`) 포인트 컬러를 사용하여 편안한 느낌을 줍니다.

##  사용 기술 (Tech Stack)

*   **HTML5:** 웹 페이지의 구조 설계 및 시맨틱 태그 활용
*   **CSS3:** 스타일링, 레이아웃(Row/Column), 반응형 디자인 구현
