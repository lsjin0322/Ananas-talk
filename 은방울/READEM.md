## 📖 소개

**은방울의 모험 2**는 HTML5 Canvas 기반의 2D 사이드스크롤 플랫폼 게임입니다.
플레이어는 작은 은방울꽃 캐릭터 **릴리**가 되어 총 6개의 스테이지를 모험하며, 적을 물리치고 아이템을 수집하고 NPC와 대화하며 세계를 치유해 나갑니다.

별도 설치 없이 브라우저에서 바로 실행되며, PWA로 모바일 홈 화면 추가도 지원합니다.

---

## 🎮 게임플레이

### 조작 방법

| 키 | 동작 |
|---|---|
| `←` / `→` | 좌우 이동 |
| `↑` | 점프 |
| `↑↑` (연속 2회) | 더블 점프 |
| `Z` | 공격 (마법 투사체 발사) |
| `X` | 씨앗 심기 |
| `C` | 월드맵 열기 |
| `Enter` | NPC 대화 |

---

## 🗺 스테이지 구성

총 6개의 스테이지로 구성됩니다.

| 스테이지 | 유형 | 설명 |
|---|---|---|
| 1~4 | 일반 | 플랫폼 탐험, 아이템 수집, 퍼즐 |
| 5 | 보스 | 강력한 보스전 |
| 6 | 비밀 | 조건 달성 시 해금되는 숨겨진 스테이지 |

스테이지 클리어 시 **별점(⭐ 0~3)**이 부여되며, 수집 아이템 및 퀘스트 완료 여부에 따라 달라집니다.

---

## ✨ 주요 기능

- **6종 아이템**: 씨앗 🌱, 종 🔔, 동전 🪙, 별 ⭐, 회복 아이템, 방어막 🛡️
- **4종 적**: 덩굴 몬스터(vine), 정령(spirit), 골렘(golem), 보스(boss)
- **트랩 시스템**: 폭탄, 스파이크, 스피드 패드, 얼음 트랩
- **씨앗 심기**: 특정 구역에 씨앗을 심어 식물을 성장시키는 퍼즐 요소
- **NPC 대화**: 스테이지에 배치된 NPC와 대화하며 스토리 진행
- **카드 매칭 퍼즐**: 일부 스테이지에서 미니게임 등장
- **월드맵**: 스테이지 선택 및 클리어 현황 확인
- **코스튬 시스템**: 기본 / 보라 / 파랑 3종 코스튬 지원
- **퀘스트 패널**: 실시간 퀘스트 진행 상황 표시
- **PWA 지원**: 모바일 홈 화면 추가 및 오프라인 플레이 가능

## 🚀 실행 방법

### 로컬 실행 (Live Server 권장)

```bash
# VS Code Live Server 포트 기본값: 5501
# index.html을 Live Server로 열기
```

1. 이 저장소를 클론합니다.
2. VS Code에서 `index.html`을 열고 **Go Live** 버튼을 클릭합니다.
3. 브라우저에서 `http://localhost:5501`로 접속합니다.

> ⚠️ `file://` 프로토콜로 직접 열 경우 Service Worker가 동작하지 않을 수 있습니다. 로컬 서버를 통해 실행하는 것을 권장합니다.
